.. py:class:: CtxInterface

   Bases: :py:obj:`Protocol`


   Minimal interface that context objects are expected to satisfy.

   These objects are meant to allow for backward computations of gradients and to warm-start
   successive forward computations. See `AcadosDiffMpcCtx` for an example of a concrete
   implementation.

   :ivar status: The status of the solver after the forward pass. `0` indicates
                 success, non-zero values indicate various errors.
   :vartype status: array of ints
   :ivar log: Statistics from the forward solve containing info like success rates
              and timings, if any.

   :vartype log: dict, optional


   .. py:attribute:: log
      :type:  dict[str, float] | None


   .. py:attribute:: status
      :type:  numpy.ndarray
