.. py:function:: add_common_args(parser: argparse.ArgumentParser, *, has_controller: bool, has_with_val: bool, wandb_group_default: str) -> dict[str, Any]

   Add common command-line arguments shared across run scripts.

   :param parser: The argument parser to add arguments to.
   :param has_controller: Whether to add controller-related args (--controller, --reuse-code*).
   :param has_with_val: Whether to add validation-related args (--with-val, --ckpt-modus).
   :param wandb_group_default: Default value for --wandb-group.

   :returns: A dict mapping group names ("run", "eval", "wandb") to the argument groups,
             so the caller can add script-specific args to the same groups.

.. py:function:: default_output_path(seed: int, tags: collections.abc.Iterable[Any] | None = None) -> pathlib.Path

   Return the default path to store experiment outputs, such as logs.

   Based on the provided seed and tags, a directory name is created.

   :param seed: The RNG seed used for the experiment.
   :param tags: Optional iterable of tags to include in the directory name.

   :returns: A `Path` to the default output directory.


.. py:function:: init_run(trainer: leap_c.trainer.Trainer[leap_c.trainer.TrainerConfigType, leap_c.trainer.CtxType], cfg, output_path: str | pathlib.Path) -> None

   Init function to run experiments.

   If the output path already exists, the run will continue from the last checkpoint.

   :param trainer: The trainer for the experiment.
   :param cfg: The configuration that was used to create the experiment.
   :param output_path: Path to save output to.

   :returns: The final score of the trainer.
