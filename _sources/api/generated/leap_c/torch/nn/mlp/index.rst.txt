.. py:class:: Mlp(input_sizes: int | Iterable[int], output_sizes: int | Iterable[int], mlp_cfg: MlpConfig)

   Bases: :py:obj:`torch.nn.Module`


   A base class for a multi-layer perceptron (MLP).

   The MLP includes a configurable number of layers and activation functions.

   :ivar mlp: The MLP model. Is `None` if no hidden layers were set in the config (see
              `MlpConfig.hidden_dims`), in which case the parameter tensor `param` is set instead.
   :ivar param: A parameter tensor with the given output size. Is `None` if hidden layers were set in
                the config, in which case the MLP model `mlp` is set instead.



   .. py:method:: forward(*x: torch.Tensor) -> tuple[torch.Tensor, Ellipsis]

      Forward the input throught the neural network.

      :param \*x: Input tensors. Will be concatenated along the last dimension. Must have shape
                  `(batch_size, input_size_i)` for each input tensor `i`.

      :returns: A tuple of output tensors, one for each output size specified in the constructor.



   .. py:attribute:: mlp
      :type:  torch.nn.Module | None


   .. py:attribute:: param
      :type:  torch.nn.Parameter | None


.. py:class:: MlpConfig

   Configuration for a multi-layer perceptron (MLP).

   :ivar hidden_dims: A sequence of integers representing the sizes of the hidden layers. If `None`,
                      no hidden layers will be used, and the MLP will be replaced with a parameter tensor with
                      the given output size.
   :ivar activation: The activation function to use in the hidden layers.
   :ivar weight_init: The weight initialization method to use for the hidden layers. If `None`, no
                      initialization will be applied.
   :ivar batchnorm: Whether to use batch normalization between layers.

   .. py:attribute:: hidden_dims
